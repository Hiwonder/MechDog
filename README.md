## Product Overview

MechDog is an advanced quadruped robot developed by Hiwonder. Featuring high-precision servo motors and intelligent control systems, MechDog can perform various complex movements and gaits. With support for multiple programming platforms including Scratch, Python, and Arduino, it provides an excellent platform for robotics education and research.

## Key Features

### Multi-Platform Programming

- **Scratch Programming** - Visual block-based programming for beginners
- **Python Programming** - Advanced programming with Python for complex projects
- **Arduino Programming** - Low-level control with Arduino IDE
- **Mobile APP Control** - Remote control via smartphone application

### Motion Control

- **Multiple Gaits** - Walk, trot, run, and custom movement patterns
- **Precise Control** - High-precision servo motors for accurate movements
- **Stable Balance** - Advanced balance algorithms for stable locomotion
- **Flexible Joints** - Multi-DOF design for complex movements
