## 产品概述

MechDog是幻尔科技开发的先进四足机器人。配备高精度舵机和智能控制系统，MechDog能够执行各种复杂的动作和步态。支持Scratch、Python和Arduino等多种编程平台，为机器人教育和研究提供了优秀的平台。

## 核心功能

### 多平台编程

- **Scratch编程** - 适合初学者的可视化积木编程
- **Python编程** - 使用Python进行高级编程和复杂项目开发
- **Arduino编程** - 使用Arduino IDE进行底层控制
- **手机APP控制** - 通过智能手机应用程序进行远程控制

### 运动控制

- **多种步态** - 行走、小跑、奔跑和自定义运动模式
- **精确控制** - 高精度舵机实现精准动作
- **稳定平衡** - 先进的平衡算法确保稳定运动
- **灵活关节** - 多自由度设计实现复杂动作
